# Begrippenlijst

| Begrip | Definitie |
| :--- | :--- |
| **Addon** | Een module die Nuvio verbindt met een inhoudsbron (films, series, anime). |
| **Plugin** | Een uitbreiding die de interface van Nuvio aanpast of specifieke lokale features toevoegt, te onderscheiden van content-scraping addons. |
| **Manifest** | Een JSON-bestand dat Nuvio vertelt wat een addon kan doen en waar de inhoud ervan zich bevindt. |
| **Debrid** | Een premium dienst die snelle, gecachte streams voor inhoud levert. |
| **Gecachte stream** | Een mediabestand dat al is opgeslagen op een Debrid-server, waardoor media direct en zonder bufferen kan worden afgespeeld, zonder afhankelijk te zijn van peer-to-peer delen. |
| **Trakt** | Een dienst van derden die bijhoudt wat je bekijkt in verschillende apps en je voortgang synchroniseert. |
| **TMDB** | The Movie Database; een door de community gebouwde database die door Nuvio wordt gebruikt om uitgebreide metadata op te halen, zoals posters, samenvattingen en castlijsten. |
| **MDBList** | Een dienst van derden die wordt gebruikt om aangepaste, dynamisch bijgewerkte inhoudscatalogi te maken op basis van specifieke beoordelingen of filters. |
| **Metadata** | De aanvullende informatie over je media, zoals samenvattingen van het plot, releasedata en afbeeldingen van miniaturen. |
| **Catalogus** | Een gecureerde rij of categorie met inhoud die rechtstreeks op het startscherm van Nuvio wordt weergegeven. |
| **Scraper** | Het deel van een addon dat het internet "afzoekt" naar stream-links. |
| **MPV** | De krachtige onderliggende videospelersoftware die Nuvio gebruikt om streams af te spelen. |
| **Libass** | Een geavanceerde ondertitel-rendering-engine die wordt gebruikt om gestileerde tekst correct weer te geven, veelgebruikt bij anime (ASS/SSA-formaten). |
| **APK / IPA** | De installatiebestandsformaten voor respectievelijk Android en iOS. |
| **Sideloading** | Het installeren van een app van buiten een officiële app store. |
| **Remux** | Een onbewerkte, niet-gecomprimeerde kopie van een Blu-ray disc die de maximale video- en audiokwaliteit biedt. |
