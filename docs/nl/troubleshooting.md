# Troubleshooting Guide

Vind hieronder oplossingen voor veelvoorkomende problemen met Nuvio.

## 1. Afspeelproblemen
| Probleem | Mogelijke oplossing |
| :--- | :--- |
| **Video stottert / buffert** | 1. Zorg ervoor dat er een Debrid-dienst actief is. <br> 2. Schakel **Hardwareversnelling** in/uit bij de instellingen. <br> 3. Controleer je internetverbinding. <br> 4. Wis het cachegeheugen van de app. <br> 5. Schakel tunneled playback aan/uit. <br> 6. Wijzig je AFR-instellingen. <br> 7. Probeer een externe player. <br> 8. Schakel snelle opstart voor ondertiteling in/uit. |
| **Geen geluid / zwart scherm** | Probeer over te schakelen naar een **externe player** (VLC of MX Player) via de instellingen. |
| **"No Streams Found"** | 1. Controleer of je addons zijn ingeschakeld. <br> 2. Controleer of je Debrid-abonnement is verlopen. <br> 3. Probeer een andere inhoudsbron. <br> 4. Onlangs uitgebrachte inhoud heeft vaak nog geen bronnen. |

## 2. App & Installatie
| Probleem | Mogelijke oplossing |
| :--- | :--- |
| **"App Not Installed"** | 1. Zorg ervoor dat je de juiste versie hebt (Mobiel vs TV). <br> 2. Controleer of je voldoende opslagruimte hebt. |
| **Crasht bij het opstarten** | Wis het cachegeheugen van de app in de systeeminstellingen van je apparaat. |
| **Synchronisatieproblemen (Trakt)** | Log uit en log opnieuw in bij Trakt in de Nuvio-instellingen. |

## 3. Addon issues
| Probleem | Mogelijke oplossing |
| :--- | :--- |
| **Manifestfout** | Zorg ervoor dat de ingevoerde URL exact klopt en begint met `https://`. |
| **Addons verdwenen** | Installeer de manifest-link opnieuw. Sommige community-links veranderen regelmatig. |

---
**Nog steeds problemen?** Bekijk de [Officiële kanalen](official-links.md) voor ondersteuning vanuit de community.
