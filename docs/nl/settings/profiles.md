# Profiles

Profiles zijn een van de belangrijkste features van Nuvio. Hier zijn enkele functiekenmerken en beperkingen van profiles:

1. Accountcapaciteit: Je kunt tot 5 profiles aanmaken en beheren onder één Nuvio-account.
2. Addon overerving vs. onafhankelijkheid: Je kunt een primair profile (Primary Profile) instellen als basis. Bij het maken van extra profiles heb je de volledige controle over hun bronnen:
    - Standalone: Kies ervoor om overerving uit te schakelen, zodat elk profile zijn eigen, volledig gescheiden addons en aangepaste catalogi gebruikt.
    - PIN-beveiliging: Profiles kunnen afzonderlijk worden vergrendeld met een aangepaste pincode om privacy te waarborgen en ongeautoriseerde toegang tot specifieke inhoud of instellingen te voorkomen. Let op: als je deze optie kiest, moet je elke keer je pincode invoeren.
3. Individuele kijkgeschiedenis: Elk profile behoudt zijn eigen voortgang, geschiedenis en "Verder kijken"-rij (Continue Watching) zonder de kijkgewoonten van anderen te beïnvloeden.
4. Trakt-synchronisatie per profile: Koppel afzonderlijke Trakt-accounts aan individuele profiles om nauwkeurige, gebruikersspecifieke externe kijklijsten bij te houden.
5. Profiles kopiëren (Cloning): Kopieer addons, plugins en aangepaste collections naadloos van het ene profile naar het andere via de webgebaseerde Account Manager.
6. Synchronisatie over apparaten: Kijkvoortgang wordt automatisch gesynchroniseerd over alle ondersteunde apparaten (Android TV, Android Mobiel, etc.).
    - Let op: app-configuraties (zoals UI-instellingen) worden niet gesynchroniseerd tussen profiles of apparaten.

## Profile aanmaken

Je kunt op verschillende manieren een profile aanmaken.
1. Direct in de apps vanaf het startscherm voor profielselectie.
2. Via de Nuvio-website.

Het aanmaken van een profile op verschillende locaties biedt iets andere opties:

**Op de website Nuvio.tv**
1. Profielnaam
2. Kleurmarkering (Color Marker)
3. Aangepaste avatar
4. Optie om addons van het primaire profile te gebruiken
5. Optie om plugins van het primaire profile te gebruiken

**In de apps**
1. Profielnaam
2. Keuze uit vooraf gemaakte avatars
3. Aangepaste avatar
4. Optie om addons van het primaire profile te gebruiken
