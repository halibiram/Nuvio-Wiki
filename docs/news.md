## New Releases - 2026-07-04

### Nuvio TV (0.7.15-beta)
#### What We Fixed and Improved

We've been busy! Here's what you can look forward to:

* We can now recognize more types of video files, including playlists. This means you can enjoy your favorite shows and movies with ease.
* We've fixed some behind-the-scenes issues that could cause problems when you're logged in.
* We've added a feature to help us understand what's going on when you experience issues, so we can fix them faster.
* We've made it harder to accidentally log out, with a confirmation dialog that asks if you really want to sign out.
* We've fixed a problem with the video player that could cause it to keep trying to play a video even when it's not possible. 
* We've also gone back to the way things were before with the stream loading overlay, which should fix some flickering issues.
* We've added more options for how long you want your videos to play automatically.

### Nuvio Web (0.3.7-beta)
#### What We Fixed and Improved

We've been working hard to improve your web browsing experience. Here's what we've done:

* We've fixed a problem that was causing the screensaver to turn on during active playback, which was a real bummer.
* We've improved the way the screensaver works, so it no longer gets in the way of your watching experience.
* We've also adjusted the subtitle delay behavior, so it's now more consistent across different devices.
* We've fixed a problem with subtitle delay persistence, so each time you start a new playback session, everything is fresh and reset.
* We've made some tweaks to the way the Back button works on Samsung Tizen devices, so it's more responsive and intuitive.
* We've fixed some issues with embedded subtitles on older webOS engines and Samsung Tizen devices.
* We've improved the way the subtitle selector works on older Samsung Tizen devices, so it's more stable and user-friendly.
* We've adjusted the player modal layout and legacy Tizen safe-area spacing to reduce clipping and make everything look more polished.
* We've also improved the way the Samsung Tizen installer works, so it's more efficient and effective.
* We've added a fallback mechanism for the Samsung Tizen installer, so it can continue even if there are issues with the shell setup commands.
* And finally, we've documented a manual workaround for affected devices, so you can still use Nuvio TV even if you have issues.
