# Windows Installation

1. **Download the installer.** Grab the latest [.msi installer from the official GitHub releases](https://github.com/NuvioMedia/NuvioDesktop/releases/latest).
2. **Launch the setup.** Double-click the downloaded file to run the installer.

> [!WARNING]
> **Windows SmartScreen Warning:** Because this is a new or unsigned release, Windows might display a "Windows protected your PC" popup. This is a false positive. Click *More info* and then *Run anyway* to proceed safely.

3. Follow the on-screen setup prompts.
4. **Grant permissions.** When the Windows User Account Control (UAC) prompt appears asking for permission, click **Yes** to allow the installation.
5. Click **Finish** to complete the setup and launch Nuvio.

You're all set! Things can be this easy when apps are allowed to be downloaded freely, not behind closed ecosystems.
